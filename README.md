# Claude Code in Production — Sample Code

Official sample code repository for **Claude Code in Production: 65 Automation Patterns for Hooks, Sub-Agents & CI/CD** by Yosuke Morikawa.

## About This Repository

This repository contains the configuration files, scripts, and agent definitions for all 65 patterns explained in the book. Browse by part number to find the sample code for each pattern.

## Repository Structure

| Directory | Patterns | Topics |
|-----------|----------|--------|
| [part1-design/](./part1-design/) | P01–P10 | CLAUDE.md, Rules, Skills |
| [part2-hooks/](./part2-hooks/) | P11–P22 | Hooks — complete 17-event catalog |
| [part3-agents/](./part3-agents/) | P23–P31 | Sub-agent design and implementation |
| [part4-cicd/](./part4-cicd/) | P32–P39 | CI/CD pipeline automation |

| [part7-workflows/](./part7-workflows/) | P57–P65 | Production workflow patterns |

## How to Use

Each directory contains ready-to-use configuration files and scripts.
Copy the files you need into your project and adapt them to your environment.

Most scripts and configurations are drop-in compatible.
Check the comment at the top of each file for pattern references (e.g., `# P14: ...`).
